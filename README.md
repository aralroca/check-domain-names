## Requeriments

## Getting started

1. Clone the repository
2. Execute `$ npm install`
3. Edit `names.json` adding the names that you want to check.
4. Execute `$ node app.js io`. Add at the end `com` if you want to check `.com`, `io` to `.io`, etc.
